# 미디어 자료 폴더

각 프로젝트 스크린샷/GIF를 여기에 넣고 `index.html`의 placeholder를 교체하면 됩니다.

| 프로젝트 | 현재 상태 | 교체 방법 |
|---|---|---|
| WES | `🏝️` placeholder | 스샷을 `wes.jpg`로 넣고, `index.html`의 `.media-placeholder` div를 `<img src="assets/img/wes.jpg" alt="WES">`로 교체 |
| NetworkProject1 | YouTube 썸네일 자동 표시 | 별도 스샷 원하면 동일 방식 |
| LeagueOfLegends-Clone | YouTube 썸네일 자동 표시 | 별도 스샷 원하면 동일 방식 |

권장: 가로 16:9, 너비 800px 이상, jpg/webp.
